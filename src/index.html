  • A project by Pietro Albini
  • Source code

About Persuade

Persuade is a simple presenter console you can use when you give talks. It allows you to have the same experience office suites provides, but with just a PDF file.

I made this because when I give talks all I take with me is the PDF file with all the slides (for compatibility purposes), and I was missing the presenter console office suites have. Having all of this reachable with just a browser is invaluable.

By the way, all the source code is publicly released.
